# MCSL

MCSL是一个跨平台的我的世界Java版服务器启动器。
## 下载

[下载最新版本](https://github.com/MCSL-V/MCSL/releases/latest)

## 运行

### Windows

1. 解压下载的压缩包到任意目录
2. 双击运行`MCSL.exe`
3. 等待启动完成

### macOS

1. 解压下载的压缩包到任意目录
2. 打开终端，进入到解压后的目录
3. 输入`./MCSL`并回车
4. 等待启动完成

### Linux

无Linux版本。

# 介绍

MCSL是一个跨平台的我的世界Java版服务器启动器。它可以帮助你快速启动我的世界Java版服务器，并提供一些便利的功能。

## 功能

- 自动下载服务器
- 自动配置服务器
- 自动启动服务器
- 服务器控制台
- 服务器状态监控
- 服务器配置文件编辑器
